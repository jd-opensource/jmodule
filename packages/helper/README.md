JModule 接入辅助工具

该包作为JModule扩展的一部分，提供了一些满足常规需求的代码片段，用于宿主应用的建设、子应用的接入。

## 对于宿主应用
1. 对于完整子应用的类型声明函数, 见于 host/type.app
2. 对于 基于Vue2动态路由的子应用的类型声明函数, 见于 host/type.routes.vue2

## 对于子应用
1. 对于 vue2 动态路由类型子应用的接入代码封装。

## 安装

```bash
npm i @jmodule/helper
```
